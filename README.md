# Tea Leaf Disease Detection

## Overview

Tea Leaf Disease Detection is a deep learning project aimed at identifying various diseases affecting tea leaves using Convolutional Neural Networks (CNNs). This project leverages image classification techniques to distinguish between healthy and diseased leaves.

## Dataset

The dataset used for this project consists of images of tea leaves categorized based on different diseases. The dataset can be accessed and downloaded from Kaggle:

[Identifying Disease in Tea Leafs Dataset](https://www.kaggle.com/datasets/shashwatwork/identifying-disease-in-tea-leafs/code)

## Features

- **Input:** Images of tea leaves (Healthy/Diseased)
- **Output:** Predicted category of the tea leaf
- **Model Used:** CNN-based deep learning model
